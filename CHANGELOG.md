# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0

- Initial public release.
- Flutter Web integration for StatCounter.
- Automatic loading of `counter.js` with initial pageview.
- SPA virtual pageview tracking via `Statcounter.track()`.
- Consent-aware loading:
  - CookieYes adapter
  - Cookiebot adapter
- Modern web interop using `package:web` and `dart:js_interop`.
